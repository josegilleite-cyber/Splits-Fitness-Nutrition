### Method 1: Using Android Emulator/Device

1. **Start the app** on Android device or emulator
2. Navigate to each screen you want to capture
3. **Take screenshot:**
   - Emulator: Click camera icon in emulator toolbar
   - Physical device: Power + Volume Down buttons
4. Screenshots will be saved to device

### Method 2: Using Expo Development Build

1. Run `npm start` and open on Android device
2. Navigate through screens
3. Take screenshots with device buttons
4. Transfer via USB or cloud

### Method 3: Using React Native Screenshot Tools

```bash
npm install --save-dev react-native-view-shot
```

### Color Palette

### Text Overlays
- **Font:** San Francisco, Roboto, or Inter
- **Size:** 40-60px for headlines, 24-32px for subtext
- **Color:** White on dark backgrounds, dark on light
- **Position:** Bottom third or top third
- **Shadow:** Subtle drop shadow for readability

### Phone Frames
Use mockup tools to add realistic device frames:
- Google Pixel frame (clean, recognizable)
- Samsung Galaxy frame
- Generic Android frame

### Consistency
- Same overlay style across all screenshots
- Consistent text positioning
- Same color scheme
- Similar content depth

## ğŸš€ Quick Start Example

### DIY Simple Version (No design tools needed):

1. **Take clean screenshots** of each screen
2. **Upload to Canva** (free account)
3. **Choose phone mockup template**
4. **Drag screenshots into phone frames**
5. **Add text boxes** with feature descriptions
6. **Download as PNG** (1080x1920 or 1440x2560)
7. **Upload to Google Play Console**

### Professional Version:

1. Take high-res screenshots (1440x2560)
2. Open in Figma or Photoshop
3. Add gradient overlays on bottom third
4. Add white text with drop shadow
5. Place in phone frame mockup
6. Add app icon and branding elements
7. Export optimized PNGs
