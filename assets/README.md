# Asset Files Needed

This app requires the following asset files in the `assets/` directory:

## Required Files:
- `icon.png` - App icon (1024x1024 pixels)
- `splash.png` - Splash screen image (1242x2436 pixels)
- `adaptive-icon.png` - Android adaptive icon (1024x1024 pixels)
- `favicon.png` - Web favicon (48x48 pixels)
- `notification-icon.png` - Notification icon (96x96 pixels, white on transparent)

## Placeholders:
For development purposes, you can use solid color placeholder images or the default Expo assets.

To generate these assets, you can:
1. Use an image editor like Figma, Photoshop, or Canva
2. Use online tools like https://www.appicon.co/
3. Use the `expo-asset-generator` package

## Icon Design Suggestions:
- Use a barbell or dumbbell icon
- Include nutrition elements (apple, plate, etc.)
- Use colors: Blue (#2196F3) and Green (#4CAF50)
- Keep it simple and recognizable
